2、每次写博客的关键命令
在cmd中切换到博客文件夹后：hexo new post “博客名”
在source->_posts下看到新创建的.md文件
生成及部署文章之前，需要安装一个扩展：npm install hexo-deployer-git –save
用编辑器编辑好文章，生成部署命令：hexo d -g
更换主题（例换成hexo-theme-jsimple）:git clone https://github.com/tangkunyin/hexo-theme-jsimple.git themes/jsimple(之后再修改_config.yml中的 theme：jsimple)
